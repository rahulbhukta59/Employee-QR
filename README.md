# Employee QR Code Generator

A simple Python mini project that generates unique QR codes for employees by encoding their details into scannable image files for easy identification and management.

## Project Overview

The Employee QR Code Generator is a Python-based application that creates QR codes for employees using basic information such as Employee ID, Name, Department, and Contact Details. These QR codes can be scanned to quickly retrieve employee information, making the system useful for identification, attendance tracking, and access control.

## Features

- Generates unique QR codes for each employee
- Encodes employee details into QR format
- Saves QR codes as image files
- Simple and beginner-friendly Python project
- Easy to modify and extend

## Technologies Used

- Python  
- `qrcode` library  
- `Pillow` (PIL)
